# EasyANOVA
Aplicación en R y Shiny para el análisis de varianza

EasyANOVA es una aplicación desarrollada en R y Shiny que permite realizar análisis de varianza (ANOVA) de forma rápida, guiada e intuitiva, sin necesidad de escribir código complicado o ser experto en R.

Basta con cargar un archivo de datos y seleccionar las variables de interés para obtener gráficos, estadísticos descriptivos y pruebas de comparación de medias listas para interpretar y reportar.

## Como exportar tus datos de Excel a CSV
1. Abre tus datos en Excel
2. Da click en Archivo > Guardar como > CSV UTF-8 (Delimitado por comas) (*.csv) > Guardar
4. De preferencia usa nombres de archivos sin espacios

## Nota:
Recuerda que para realizar un ANOVA se deben de cumplir los siguientos supuestos:

1. Normalidad de los datos
2. Homogeinidad de varianzas
3. Independencia de las observaciones
